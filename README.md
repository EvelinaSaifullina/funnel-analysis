# funnel-analysis
# Анализ воронки продаж (SQL, Python): поиск точки разрыва

## Результаты

| Этап | Пользователи | Конверсия от home | Конверсия от предыдущего |
|------|-------------|-------------------|--------------------------|
| home | 90 400 | 100% | — |
| search | 45 200 | 50% | 50% |
| payment | 6 030 | 6.67% | 13.34% |
| confirmation | 452 | 0.5% | 7.5% |

**Самый слабый этап (точка разрыва):** payment → confirmation (7.5% конверсии).

## Гипотеза

**Проблема:** пользователи доходят до страницы оплаты, но не завершают покупку.

**Решение:** упростить форму оплаты — добавить автозаполнение адреса, убрать лишние поля, добавить кнопку "Купить в 1 клик".

**Ожидаемый эффект:** +10 п.п. к конверсии → число покупок вырастет с 452 до ≈1055 (+133%).

## Другие наблюдения

- home → search (50%): половина пользователей не начинает поиск
- search → payment (13.34%): самая большая абсолютная потеря (39 170 пользователей)
